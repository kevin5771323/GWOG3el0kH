# 项目简介

## 核心功能
- 基于YOLOv8的水洼实例分割检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图

## 适用场景
- 水洼检测与识别
- 实时视频监控中的目标检测
- 批量图片分析
- 科研实验与教学演示
- 环境监测相关应用

## 技术特点
- 基于Python 3.9开发
- 使用PyCharm+Anaconda或VSCode+Anaconda作为开发环境
- 集成OpenCV、PyQt5、Ultralytics、PyTorch等技术栈
- 提供完整的用户界面，操作简便
- 包含1000张标注数据集，支持['flaque']一类目标检测
- 已训练好模型，配置成功环境后可直接使用
